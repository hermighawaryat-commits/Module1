# Day 01 Notes - Terminal & Git Revision

## Top 5 Terminal Commands
1. `pwd` - Print Working Directory: Displays the full path of the current folder you are in.
2. `ls -la` - List All Contents: Shows all files and directories, including hidden ones (`.files`), with details.
3. `cd <folder>` - Change Directory: Moves into the specified folder (`cd ..` moves up one level).
4. `mkdir <name>` - Make Directory: Creates a new folder with the given name.
5. `rm` - Remove: Deletes a file immediately without moving it to a trash/recycle bin.

## Top 3 Git Commands
1. `git status` - Displays the state of the working directory and staging area (shows modified/untracked files).
2. `git add .` - Moves changes from the working directory into the staging area, preparing them to be snapshot.
3. `git commit -m "message"` - Saves a permanent snapshot of the staged changes into the local repository history.
